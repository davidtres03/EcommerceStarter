## ?? **Security Layers**

### **Layer 1: Network Perimeter**
```
INTERNET
    ?
?????????????????????????????????
?  Router / Firewall            ?
?  ? Port 80/443 ? Host only    ?
?  ? All other ports blocked    ?
?  ? Optional: Port 22 if remote?
?????????????????????????????????
    ?
HOME NETWORK
```

### **Layer 2: Host Firewall**
```
Windows Firewall (Host Machine)
???????????????????????????????????????????
?  Inbound Rules:                         ?
?  ? Port 80  - HTTP    - All sources    ?
?  ? Port 443 - HTTPS   - All sources    ?
?  ? Port 22  - SSH     - 192.168.1.50   ?
?  ? Port 1433 - SQL    - BLOCKED        ?
?  ? All others         - BLOCKED        ?
?                                         ?
?  Outbound Rules:                        ?
?  ? All - ALLOWED                       ?
???????????????????????????????????????????
```

### **Layer 3: Application Isolation**
```
IIS Application Pool
???????????????????????????????????????????
?  Identity: ApplicationPoolIdentity     ?
?  ? Isolated process                    ?
?  ? No managed code                     ?
?  ? Minimal file system permissions     ?
?  ? Cannot access other apps            ?
???????????????????????????????????????????
```

### **Layer 4: Database Security**
```
SQL Server
???????????????????????????????????????????
?  ? Localhost access only               ?
?  ? Encrypted connections required      ?
?  ? Dedicated app user (not sa)         ?
?  ? Minimal permissions granted         ?
?  ? No direct internet access           ?
???????????????????????????????????????????
```

### **Layer 5: Authentication**
```
Application Security
???????????????????????????????????????????
?  ? ASP.NET Core Identity               ?
?  ? Role-based authorization            ?
?  ? Strong password requirements        ?
?  ? Azure Key Vault for secrets         ?
?  ? HTTPS enforced                      ?
?  ? Security headers                    ?
???????????????????????????????????????????
```

| Port | Protocol | Source | Destination | Action | Purpose |
|------|----------|--------|-------------|--------|---------|
| 80 | TCP | Any | Host | ? Allow | HTTP (redirects to HTTPS) |
| 443 | TCP | Any | Host | ? Allow | HTTPS (main site) |
| 22 | TCP | 192.168.1.50 | Host | ? Allow | SSH (dev machine only) |
| 22 | TCP | Any other | Host | ? Block | SSH (security) |
| 1433 | TCP | Any | Host | ? Block | SQL (not exposed) |
| 3389 | TCP | LAN only | Host | ?? Optional | RDP (if needed) |
| * | * | Host | Any | ? Allow | Outbound (updates, GitHub) |
| * | * | Any | Host | ? Block | Default deny inbound |
