## ?? Key Principles

### 1. **Never Touch Production Directly**
```
? Direct production DB edits
? Testing on live site
? Deploying untested code

? Test on EcommerceStarter_Test
? Verify all tests pass
? Deploy through proper process
```

### 2. **Backup Before Everything**
```
Before deployment  ? Backup
Before migration   ? Backup
Before schema change ? Backup
Friday afternoon   ? DEFINITELY backup!
```

### 3. **Test, Then Test Again**
```
Local testing      ?
Test database      ?
Staging (if exists) ?
Production         ? (with backups!)
```

### 4. **Keep Open-Source Clean**
```
No production secrets ?
No customer data      ?
No personal info      ?
Generic branding      ?
MIT Licensed          ?
```
